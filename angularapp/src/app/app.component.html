Thank you for completing our recruitment home exercise.

Overview

We would like you to create a simple book display page and form to display book information.

Project

We've setup a template project for you. It includes:

  • An Angular frontend embedded in a .Net web application (All UI components should be implemented in Angular)
  • The beginning of an API - A book API controller exists for you

Expected completed solution

When you have completed this exercise, you should remove these instructions. There should be a very simple library management application in their stead.

Please remove the home component and replace it with your own, or use routing to create a link to your library management app page.

The application needs to:

  • Display a list of books retrieved from https://www.googleapis.com/books/v1/volumes?q=trees
  • A book from the catalogue can be chosen and loaded into a basic editor.
  • Display a form with a few fields, such as ID, Title, and Author, and display the book detail in the form when the edit button is clicked
  • There is no need to update or introduce functionality.
  • Please give careful consideration to the final product's aesthetics.
